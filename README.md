# globalimport-web
Trabajo de página web respecto a un comercio electrónico.

El repositorio va dirigido a la empresa "GlobalImportal S.A.", la cual se dedica a la distribución e importación de productos electrónicos, así como también indumentaria y artículos hogareños.
Es así como la empresa ha optado por planear la producción de una página web dedicada a sus servicios.
